A Python-based hand gesture system to control browser tabs using OpenCV, MediaPipe, and PyAutoGUI

# Hand Gesture Browser Control

Control your browser tabs using real-time hand gestures via your webcam.

## 🚀 Features
- ✋ Open Palm → Next Tab  
- ✊ Fist → Previous Tab  
- Real-time gesture detection using webcam  
- Fast and responsive control  

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## 📦 Installation

1. Clone the repository:
   https://github.com/akchansh/hand-gesture-browser-control.git

2. Install dependencies:
   pip install -r requirements.txt

   
## ▶️ Usage

Run the script: python gesture_control.py

Or double-click: start_gesture.bat


## ⚠️ Notes
- Ensure your webcam is working
- Good lighting improves gesture detection accuracy

## 📁 Project Structure
gesture_control.py # Main script
gesture_model.tflite # Gesture model
start_gesture.bat # Quick start script
